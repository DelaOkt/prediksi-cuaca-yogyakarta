Prediksi Cuaca Yogyakarta

Prediksi cuaca ini berfokus pada curah hujan (RR). Dashboard ini menyajikan hasil prediksi curah hujan harian di Yogyakarta tahun 2024 menggunakan model Machine Learning LSTM dengan bobot ekstrem. Data diperoleh dari BMKG Stasiun Klimatologi Yogyakarta (2020–2024).

Klasifikasi Curah Hujan
☀️ Tidak Hujan: 0 mm
🌦️ Hujan Ringan: 0 - 20 mm
🌧️ Hujan Sedang - Lebat: 20 - 49.9 mm
⚠️ Cuaca Ekstrem: ≥ 50 mm
